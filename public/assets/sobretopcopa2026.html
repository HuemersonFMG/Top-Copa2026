Sobre o Top Copa 2026

Projeto web para acompanhamento da Copa do Mundo FIFA 2026.

🏆 Visão Geral

Projeto Estácio - Aplicando conhecimentos do Curso

Api Copa 2026 densenvolvida para a empresa Top Cestas de Alimentos - Cortesia.

O Top Copa 2026 é um site desenvolvido para acompanhar os jogos, placares e classificação da Copa do Mundo FIFA 2026 de forma simples, visual e responsiva.

A aplicação exibe os grupos da competição, os jogos de cada seleção, o próximo jogo do Brasil e os resultados atualizados automaticamente através de uma API esportiva externa.

⚽ Principais Funcionalidades

Classificação

Tabelas completas dos grupos com pontos, jogos, vitórias, empates, derrotas e saldo de gols.

Jogos

Lista de partidas com data, horário, seleções, bandeiras, placar e status do jogo.

Brasil

Destaque especial para o próximo jogo da Seleção Brasileira e acompanhamento da campanha.

🔄 Atualização Automática

O servidor consulta periodicamente a API Football-Data.org, processa as informações dos jogos e grava os dados em cache local.

Dessa forma, o site permanece rápido, evita excesso de chamadas externas e mantém os placares atualizados durante a competição.

Node.js Express Football-Data.org PM2 Cloudflare Tunnel

🛠 Tecnologias Utilizadas

O projeto foi desenvolvido com HTML5, CSS3, JavaScript, Node.js e Express. Em produção, a aplicação roda com PM2 e é publicada por meio do Cloudflare Tunnel.
